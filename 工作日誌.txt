## 修改紀錄與功能更新

### 1. 專案基礎設定與導覽
- **[新增]** 專案本地預覽功能，透過 `npm run dev` 啟動 Vite 開發伺服器。
- **[更新]** 更新安全對位框 (Safety Guides) 的底圖網址為專用版本。

### 2. 「小檔案 (Profile)」版型深度修改
- **[新增]** 文字自動換行 (`autoWrap`) 屬性：
  - 支援標題與摘要文字在超出設定寬度時自動換行，避免字體變形。
- **[新增]** 自動載入底圖機制：
  - 點擊「小檔案」版型時，會自動載入特定的背景裝飾底圖。
- **[調整]** 版面結構重整：
  - **頂部**：中心大標題。
  - **左側**：圖片顯示區域。
  - **右側**：兩組「小標題 + 內文摘要」組合。
- **[新增]** 無底圖動態樣式（適應性渲染）：
  - 當使用者**關閉底圖**時，小檔案版型會自動切換樣式：
    - 主標題自動加上「藍邊白字」特效（隨主題色變化）。
    - 小標題與內文自動襯托「雙框版型風格」的藍色/白色背景 Bar。
    - 所有 Bar 寬度與內文寬度 (800px) 對齊一致。

### 3. 對位框 (Safety Guide) 操作介面優化
- **[新增]** 右上角控制面板：
  - **功能開關 (Toggle)**：可隨時開啟或隱藏對位框。
  - **透明度拉桿 (Slider)**：支援 0% - 100% 透明度調節，預設值為 50%。
- **[調整]** UI 佈局微調：
  - 將透明度拉桿置於開關左側，並固定開關位置於右上角導覽列。

### 4. 版本控制與備份
- **[新增]** Git 儲存庫初始化。
- **[新增]** GitHub 遠端備份：
  - 目前已推送至 `main` 主分支。
- **[新增]** 發布分支建立：
  - 建立並推送至 `TVBS_AICG_release` 分支，作為階段性正式版本存檔。

### 5. 全新畫布式圖片裁切與編輯系統
- **[重大更新]** 移除彈出式裁切視窗 (Crop Modal)，改為直接在主預覽畫布進行交互式編輯。
- **[新增]** 圖片遮罩 (Masking) 邏輯：
  - 圖片置入時自動採取 `cover` 模式填滿容器。
  - **內部操作**：滾輪放大原圖、左鍵拖動調整裁切範圍。
  - **外部操作**：選取後顯示 5px 半透白框，用於移動整體容器位置。
  - **右鍵功能**：快速復原圖片至初始的「滿版置中」狀態。
- **[新增]** 邊緣像素汲取延伸 (Edge Extension)：
  - 當圖片縮得比遮罩小時，背景會自動生成模糊化的原圖邊緣進行填補，避免露出白色空白或底色。
- **[新增]** 強制等比例縮放 (Ratio Constraint)：
  - **圖片圖層**與**文字排版圖層**在調整外框大小時，會強制鎖定寬高比例，確保排版與字體不產生左右或上下壓扁的變形。
  - **自動修正**：替換圖片或調整已變形圖層時，系統會自動歸位至正確比例。

### 6. 開發參考工具
- **[新增]** 座標參考標籤 (Coordinate Guides)：
  - 在編輯模式下，元件左上方會顯示即時座標 (X, Y) 與尺寸資訊。
  - 該標籤僅供開發參考，不會被導出到成品中。

### 7. 前置關卡 (Onboarding Setup) 與智能帶入
- **[新增]** 在進入主編輯器前加入「前置作業與排版設定」關卡介面。
- **[新增]** 提供 6 種版型的視覺化選擇 (小檔案、雙框、三框、文章拉字、傷勢圖、社會 NCCG)。
- **[新增]** 支援使用者預先貼上新聞內文，系統會自動拆解第一句為「主標題」，其餘為「內文」，並自動套用至選擇的版型。
- **[新增]** 支援最多上傳 3 張圖片，進入畫布時自動對應至版型的圖片框內。
- **[優化]** 預留「傷勢圖」與「社會 NCCG」的基礎空白版型架構，供後續擴充使用。
